# wldos

#### Description
WLDOS[worldAS]是一款互联网运营平台，本仓库是wldos后端工程。
WLDOS开发平台，基于springboot实现轻量级快速开发框架，SaaS应用架构。默认支持多租户运行模式，同时支持关闭以单租户模式运行。默认单实例运行，在分布式部署方面支持融入serviceMesh架构或者传统中心化分布式架构。
