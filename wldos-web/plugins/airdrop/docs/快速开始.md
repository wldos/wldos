# 脚本神器插件 - 快速开始

## 🚀 5分钟快速上手

### 第一步：配置比特浏览器

1. **安装比特指纹浏览器**
   - 下载地址：https://bitbrowser.cn/
   - 安装并启动，确保API服务运行（默认端口：54345）

2. **创建浏览器配置文件**
   - 在比特浏览器中创建配置文件
   - 记录配置文件ID（如：`profile_001`）

3. **配置插件连接**
   ```yaml
   # application.yml
   bitbrowser:
     api:
       url: http://127.0.0.1:54345
       key: your_api_key_here
   ```

---

### 第二步：创建第一个任务

#### 方式1：通过UI创建（推荐）

1. 登录WLDOS系统
2. 进入 **脚本神器** → **自动化调度器**
3. 点击 **创建调度任务**
4. 填写信息：
   - 任务名称：`测试任务`
   - 项目类型：`通用`
   - 脚本类型：`JavaScript`
   - 脚本内容：
   ```javascript
   console.log('Hello from Airdrop Plugin!');
   await page.goto('https://example.com');
   await page.waitForTimeout(2000);
   console.log('任务完成');
   ```
   - 浏览器配置：选择 `profile_001`
   - 队列配置：
     - 并发数：1
     - 任务间隔：2秒

5. 点击 **创建** 并 **执行**

#### 方式2：通过API创建

```bash
curl -X POST http://localhost:8080/api/airdrop/scheduler/tasks \
  -H "Content-Type: application/json" \
  -d '{
    "name": "测试任务",
    "projectType": "generic",
    "scriptType": "javascript",
    "scriptContent": "console.log(\"Hello World\");",
    "profileIds": ["profile_001"],
    "queueConfig": {
      "concurrency": 1,
      "taskInterval": 2
    }
  }'
```

---

### 第三步：查看执行结果

1. 在 **执行记录** 页面查看任务执行情况
2. 查看执行日志和错误信息
3. 根据结果调整脚本和配置

## 💡 常用脚本模板

### JavaScript - 简单页面操作

```javascript
// 访问页面并点击按钮
await page.goto('https://example.com');
await page.waitForTimeout(2000);
const button = await page.$('button#submit');
if (button) {
    await button.click();
    await page.waitForTimeout(1000);
}
console.log('操作完成');
```

### Python - 数据处理

```python
# 处理数据并输出
import json
data = {"task": taskName, "timestamp": int(time.time())}
print(json.dumps(data))
```

### Shell - 文件操作

```bash
# 创建文件并写入内容
mkdir -p /tmp/airdrop
echo "Task ID: $taskId" > /tmp/airdrop/task.log
echo "执行完成"
```
